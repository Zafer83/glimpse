STEP 4 - AVAILABLE SLIDEV LAYOUTS (use the correct ones):

- layout: cover — Cover page with title
- layout: end — Final "Thank You" page
- layout: default — Standard content slide
- layout: center — Content centered on screen
- layout: intro — Introduction with title, description, author
- layout: section — Section divider between topics
- layout: statement — Bold statement/affirmation
- layout: fact — Prominent fact or data point
- layout: quote — Quotation with prominence
- layout: image-right — Image right, content left
- layout: image-left — Image left, content right
- layout: image — Full-screen image
- layout: two-cols — Two columns (use ::right:: separator)
- layout: two-cols-header — Header spanning both columns (use ::left:: and ::right::)
- layout: full — Full screen content
- layout: iframe — Embedded web page
- layout: none — No styling
